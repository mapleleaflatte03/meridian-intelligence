1-Command Install

Install Meridian Loom locally and get the Meridian execution runtime under your own home directory.

Launch installer

curl -fsSL https://raw.githubusercontent.com/mapleleaflatte03/meridian-loom/main/scripts/install.sh | bash

The installer pulls a release binary first, provisions $HOME/.local/share/meridian-loom, and only falls back to a local Rust build when a matching binary is unavailable.
